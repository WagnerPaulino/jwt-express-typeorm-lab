# Lab jwt e typeorm

Steps to run this project:

1. Run `npm i` command
2. Insert the first admin user `npm run migration:run` command
3. Run `npm start` command

Create a migration
```
Run `npx typeorm migration:create -n CreateAdminUser` command
```
